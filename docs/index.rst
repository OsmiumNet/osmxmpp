Introduction
============

OsmXMPP is a Python library for XMPP protocol.
It's designed to be used with the `OsmiumNet XML library <https://github.com/osmiumnet/osmxml>`_.


Installation
------------

To install the library, run the following command:

.. code-block:: bash

    pip install osmxmpp
